# Hip-Hop

### 힙한 언어 Hip-Hop

색을 변수로 사용하는 세상에 하나밖에 없는 언어
각각의 코드가 고유한 색을 가지는 언어

이해를 하지 못한다면 그것은 힙하지 않기 때문

힙해지고 싶어? 그럼 한번 해봐!

### 언어 설명

변수는 색으로 지정합니다.

단항연산자의 경우 첫번째 글자의 색을 사용합니다.
이항연산자의 경우 첫번째 글자의 색과 두번째 글자의 색을 연산하여 첫번째 글자 색에 결과를 대입합니다.
